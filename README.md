# Calculadora – Método das Direções (UFPE)

Este aplicativo permite carregar uma planilha de campo, validar, calcular e gerar tabelas de ângulos e distâncias.

## Arquitetura
- app.py → fluxo do Streamlit
- processamento.py → conversões angulares e cálculos
- utils.py → funções auxiliares
